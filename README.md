# AccessHandicap
Un site genere par ChatGPT qui n'etait pas accesible aux personnes handicape.

Lors de mes cours, j'ai pu utiliser plusieurs moyens pour rendre mon site accessible, comme par exemple le fait de d'utliser WAVE (web accessibility evaluation tool) et WCAG (contrast checker). 

Suite aux informations de ces extensions, j'ai pu rendre mon site accessible notamment avec un niveau AA.
